# todosvc2 
* 할일 목록 서비스 using mongodb
---------------------------------------
## 실행 방법
* mongodb 기동
~~~
예) dao/tododb.js의 설정을 적절하게 변경하세요.
./bin/mongod --dbpath data/db --port 27017
~~~

* 의존성 설치
~~~
npm install
~~~
* 개발 모드로 실행
~~~
npm run start:dev
~~~
* 운영 모드로 실행
~~~
npm run build
npm run start
~~~
----------------------------------------

## 사용방법 확인
